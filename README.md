# 司法・予備論文DB 南国モバイルリーダー

> 予備試験・司法試験論文素材DBを、スマホで軽く・速く・迷わず復習するための単一HTMLリーダーです。

![OGP preview](assets/ogp.png)

## これは何？

論文用の素材DBを、スマホで見やすいカード型リーダーにしたものです。  
科目別に読み、横断検索で探し、カード内で条文候補・答案順序・抗弁反論をまとめて確認できます。

## 主な機能

- 科目別ページ
- 横断検索
- 重要度フィルタ
- 条文引用ブロック
- 条文候補・漏れチェックタグ
- 論文答案化フレーム
- 3分答案化モード
- 最終QA / 監査サマリー
- スマホ向けナビゲーション
- GitHub Pages対応

```

```

## GitHub Pagesで公開する

このリポジトリはビルド不要です。  
`index.html` をGitHub Pagesで配信するだけで動きます。

詳しい手順は [`DEPLOY.md`](DEPLOY.md) を見てください。

## 注意

このHTMLは復習・検索・答案構成練習用です。  
条文引用は監査済みですが、提出答案や厳密な引用が必要な場面では、必ずe-Gov等の公式原文を最終確認してください。
